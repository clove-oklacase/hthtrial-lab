# hthtrial-lab

## 项目简介

hthtrial-lab 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面来自不同的项目或实验，彼此之间没有依赖关系，也不针对任何单一域名或网站。本仓库旨在提供一个干净、有序的存储空间，方便后续查看、分享或引用。

## 目录结构

```
hthtrial-lab/
├── pages/          # 存放所有独立 HTML 页面（可包含子文件夹）
├── assets/         # 公共资源文件（如 CSS、JS、图片等，可选）
├── README.md       # 本说明文件
└── LICENSE         # 仓库许可证（如有）
```

- `pages/` 是核心目录，每个 HTML 页面建议单独放入一个文件夹，避免文件冲突。
- `assets/` 为可选目录，用于存放多个页面共用的资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的归档文件，可直接在浏览器中打开运行。
- 页面可能包含外部资源（如 CDN 链接），但这些资源不在本仓库维护范围内。
- 归档页面不保证长期可用性或兼容性，但会尽量保持原始状态。
- 如需引用某个页面，建议直接使用仓库中的原始文件路径或下载后自行托管。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时不会逐一通知。
- 欢迎通过 Issue 提交问题或建议，但请注意本仓库不提供技术支持或使用指导。
- 提交 Pull Request 时，请确保新页面不与现有文件冲突，并尽量保持目录结构清晰。
- 仓库许可证信息请见 `LICENSE` 文件（如有），未明确许可证的页面请默认遵循仓库许可证。

## 其他

- 本仓库仅用于存档目的，不包含任何营销、推广或诱导内容。
- 所有页面内容均来自维护者个人实验或项目，与任何组织、商业实体无关。

如有疑问，请通过 Issue 联系维护者。
